These two graphs shows information about number of people using facebook over period and the reasons of using this website. In the first graph the horizontal axis corresponds to the year while vertical one refers to number of users. The second graph shows the "main reasons" in vertical axis while horizontal axis corresponds to male and percentage.
Generally speaking, the main difference between 2012 and 2013 year on the first graph is that people start using less computers and the number of people decreased. The main feature of these two graphs is that the main reason to visit facebook is to share photos. So the industry changed to mobile because it is easier to share photos with phone.
For example, this trend of using phones contains not only facebook. From march 2013 we can say that the most important device in our life is smartphone because we can do anything with that device.
