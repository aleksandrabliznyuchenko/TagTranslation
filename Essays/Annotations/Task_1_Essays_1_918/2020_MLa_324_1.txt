The chart gives information on changes in the use of social network Facebook over a one-year period from 2012 to 2013 and the main reasons for using it among people both genders.
As we can see on the first bar chart the overall use of Facebook decreased significantly during the given period. Moreover there are appeared to be some changes in the way of using Facebook. In 2012 desktop using of this social network was more popular than mobile using, while in 2013 mobile using grew upon desktop on 20 percent.
The second bar chart shows us that the most important reason for women for visiting Facebook website is sharing photos and videos. At the same time men call this reason and receiving updates in the same amounts.
To sum up, the use of Facebook fell from 2012 to 2013.
