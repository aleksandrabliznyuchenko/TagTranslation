The graphs feature information about the differences in using Facebook from 2012 to 2013 and the key reasons why it was used by men and women.
From an overall perspective, it can be mentioned that using desktop reduced over the period, while using mobile became more popular by 2013. What is more, Facebook was used to share photos or videos by women mostly.
To be exact, using desktop fail from 150 millions of users in 2012 to just under 140 millions in 2013, while using mobile increased considerably, rising from roughly 60 millions of users in 2012 to almost 100 millions people using Facebook. However, using desktop was still more popular by the end of the period.
More than a half of women used this social network to share photos and videos. The second reason for using Facebook by women was seeing funny posts, which counted for 43%. Almost two-fifths of females used it for receiving updates, while learning about ways to help others was the least popular.
Men used Facebook equally for sharing photos and receiving updates, which made up 39%. 35% of males used this social network to see funny posts, and learning about ways to help others was the least popular among males.
